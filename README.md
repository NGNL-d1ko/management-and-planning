# MaP

**MaP** (`Management and Planning`) — веб-приложение для управления задачами, проектами и личным/рабочим планированием.  
Проект разработан как дипломная работа и представляет собой единое пространство для организации работы, контроля сроков, визуализации процессов и отслеживания прогресса.

## О проекте

Приложение объединяет несколько основных сценариев управления деятельностью:

- ведение списка задач;
- работа с проектами и их прогрессом;
- Kanban-доска для визуального контроля этапов;
- календарное планирование событий;
- аналитика по выполнению задач и продуктивности;
- профиль пользователя и пользовательские настройки.

Основная идея проекта — предоставить удобный интерфейс, в котором пользователь может работать с разными уровнями планирования в одном приложении: от отдельных задач до проектных сущностей и общего обзора прогресса.

## Основные возможности

### Dashboard

- сводка по задачам;
- карточки статистики;
- быстрый просмотр активных проектов;
- мини-календарь недели;
- быстрое добавление задачи.

### Tasks

- просмотр списка задач в табличном виде;
- фильтрация по статусу, приоритету и проекту;
- поиск и сортировка;
- создание, редактирование и удаление задач;
- массовые действия;
- отметка избранных задач.

### Boards

- Kanban-доска с drag-and-drop;
- пользовательские колонки;
- перемещение задач между этапами;
- создание и удаление колонок;
- быстрый поиск по задачам на доске.

### Projects

- карточки проектов;
- статусы и приоритеты проектов;
- чек-лист прогресса;
- список участников;
- избранные проекты;
- просмотр детальной информации по проекту.

### Calendar

- календарное отображение событий;
- переключение режимов просмотра;
- создание и удаление событий;
- фильтрация по типу события;
- список событий на текущий день.

### Analytics

- графики продуктивности;
- распределение задач;
- прогресс по проектам;
- показатели эффективности;
- экспорт данных как демонстрационный сценарий интерфейса.

### Profile

- данные пользователя;
- настройки интерфейса;
- параметры уведомлений;
- экспорт и импорт пользовательских данных;
- история активности;
- демонстрационные настройки безопасности.
